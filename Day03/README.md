## 摘要

+ 样式表如何从文件的顶部到底部级联
+ 具体是什么权重,如何计算权重
+ 如何组合选择器以及定位特定元素或者元素组
+ 如何在单个元素上使用多个类来为不同的式样分层以获得更多模块化代码
+ 可在CSS中使用的不同颜色值,包括关键字,十六进制,RGB和HSL值
+ 可在CSS中使用的不同长度值,包括像素,百分比和em单位
