## 学习内容摘要

> 列表在HTML中非常常用，通常在可能不明显或不明显的地方使用。关键是尽可能在语义上使用它们，并在最适合的地方利用它们。

+ 如何创建无序，有序和描述列表
+ 如何在其他列表中正确嵌套列表
+ 如何更改列表项标记样式和位置
+ 如何使用背景图像而不是列表项标记
+ 如何水平显示或浮动列表

> 既然我们知道如何向页面添加列表，那么我们也可以在页面中添加媒体。在下一章中，我们将深入研究可嵌入媒体，如图像，音频和视频。
