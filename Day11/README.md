# 学习内容：表格

## 学习内容摘要：

> 好吧，我们现在知道如何在HTML中语义地布局表格数据，同时使CSS直观。讨论表是我们学习HTML和CSS的最后一个主要障碍，我们现在正式完成了Styles Conference网站。

> 要查看，在本课程中我们介绍了以下内容：

+ 在表格上设置边框样式的不同方法，以及不同的边框属性如何影响表格的外观
+ 如何垂直对齐表格中的文本
