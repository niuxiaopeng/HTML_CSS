## 学习内容摘要

+ 如何将背景颜色和图像添加到元素
+ CSS渐变，包括线性和径向，以及如何自定义它们
+ 如何将多个背景图像应用于单个元素
+ 新的CSS3属性，允许我们更改背景图像的大小，表面区域和原点

>添加背景颜色，渐变和图像为增强我们网站的整体设计带来了很多可能性。很快我们将讨论如何将图像（除了背景图像）和其他媒体语义添加到我们的页面。但在此之前，让我们来看看如何在语义上创建列表。
